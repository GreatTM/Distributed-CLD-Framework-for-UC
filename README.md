# Supplementary-material-for-Distributed-CLD-Framework-for-UC
For the manuscript “Yemin Wu, Shuai Lu, Wei Gu, et al., Distributed Closed-Loop Decision Framework for Unit Commitment".

1. `Supplementary Material.pdf` contains the appendices of the manuscript. Due to the 10-page limit, this material is not included in the main text at the current stage.

2. `DCLD_Supplementary_Data_2025.xlsx` contains all supplementary datasets, parameters, and configurations for the simulations presented in the manuscript.

3. `privacy_inference_analysis.pdf` provides the supplementary privacy inference analysis, including an attack-surface summary and the mathematical derivations supporting the privacy discussion for the inner ADMM iterations of the proposed distributed framework.
